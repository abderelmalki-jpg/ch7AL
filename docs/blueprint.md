# **App Name**: Souk Price

## Core Features:

- User Authentication: Secure user registration and login using Firebase Authentication.
- Product Barcode Scanner: Native barcode scanning using Capacitor to fetch product information.
- Price Contribution: Allow users to add and update prices for products, earning points for their contributions.
- AI Product Suggestion: Suggest alternative product names based on description of product to make it easier for other users to find prices. Use of AI will act as tool.
- Geolocated Hanout (Store) Display: Display nearby stores on an interactive map using Capacitor's geolocation capabilities and Firebase data.
- Reward System: Award badges to users based on their contribution levels, encouraging active participation.
- Multilingual Support: Offer the application in French, Arabic, and Darija to cater to a wider user base.

## Style Guidelines:

- Primary color: Vibrant green (#3CBC8D) to evoke freshness and value.
- Background color: Light green (#E9F8F3) for a clean, inviting feel.
- Accent color: Yellow-orange (#F2994A) for CTAs and highlights, providing a warm contrast.
- Headline font: 'Belleza' sans-serif for titles; pairs with 'Alegreya' serif for body.
- Use clear, consistent icons for navigation and actions, with a touch of local flair (e.g., souk-related icons).
- Clean, card-based layout for product listings and store information.
- Subtle transitions and animations on user interactions (e.g., scanning, price updates).
